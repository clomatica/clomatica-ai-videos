# Clomatica — Getting Started

Welcome to your AI workspace! This guide will help you understand how everything works.

> **First time?** Run `/setup` in the chat to configure your workspace. This guide will be updated with your personalized content after setup.

---

## What Is This?

This is an AI workspace that grows smarter through use. Every time you work with Claude, it can save repeatable processes as reusable skills — shortcuts your whole team can use.

- **Ask for help with any task** — writing, analysis, research, operations
- **Claude does the work with you** — using your connected tools (Slack, Notion, etc.)
- **Repeatable processes become skills** — saved as shortcuts anyone can run
- **Your workspace gets smarter over time** — the more you use it, the more it learns

### How It Works

The workspace has three parts:

1. **Context** — what the assistant knows about your company (values, processes, tools)
2. **Skills** — predefined tasks you can run with a single command (like `/setup`)
3. **Learning** — as you work, the assistant offers to save new processes as skills

---

## Two Ways to Use It

### Claude Code App (Recommended for beginners)
Open the Claude Code app — it looks like a chat window. Just type what you need.

### VS Code Extension
If you prefer VS Code, open this folder in VS Code and use the Claude Code extension panel.

Both do the same thing. Start with whichever feels more comfortable.

---

## Daily Workflow

```
Morning:   Open the workspace, ask Claude what you need
During:    Work together — Claude helps with tasks using your tools
End:       Say "save changes" so your work is backed up
```

---

## Adding Documents and Templates

Want to teach the workspace about your company? Just drop any file in the main folder and tell the assistant what it is:

- **"This is our brand guide"** → gets organized into company knowledge
- **"This is the template we use for proposals"** → gets saved as a reusable reference
- **"These are our SOPs for client onboarding"** → gets turned into a skill

You don't need to know where things go — just drop the file and describe it. The assistant organizes everything automatically.

## Need Help?

- Type anything in the chat — Claude will help
- Run `/setup` to configure your workspace
- If Claude stops responding, reload the window (Cmd+Shift+P → "Reload Window")

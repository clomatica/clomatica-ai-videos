# Clomatica — AI Workspace

This folder is Clomatica's AI workspace — a shared space where your team works with an AI assistant that learns how your company operates.

---

## Start here

1. Open this folder in the **Claude Code app** (or in VS Code with the Claude Code extension)
2. Type `/setup` in the chat and press Enter
3. The assistant takes it from there — it connects your tools and personalizes everything

Already set up? Just open the chat and ask for what you need.

---

## How it works

1. Ask the assistant for help with any task
2. It does the work with you, using your connected tools
3. At the end, it offers to save repeatable processes as shortcuts your whole team can reuse
4. Your workspace gets smarter over time

---

## Good to know

- **GETTING-STARTED.md** is your full usage guide — how everything works, troubleshooting, and a glossary of terms
- The `.claude` folder is the workspace's memory — never delete or move it
- Your credentials and private files stay on this computer — they are never uploaded
